# 《永夜熵纪》V13.0 补充协议：前沿探索与多模态扩展

基于 V12.2（永夜熵纪·自定义AI模型版）及极限性能测试补充文档，为了将这款“无后端数字文明模拟器”的纯前端潜能推向理论极致，解决超大模型分发、跨设备连续性以及沉浸式交互等长期依赖后端的痛点，特制定本 V13.0 补充协议。所有新增特性依然严格遵循“零后端服务器”与“不请求敏感额外权限”的核心理念。

---

## 4. 底层架构与生存优化（V13.0 再补充）

### 4.10 WebTorrent 驱动的 P2P 联邦大模型分发引擎

为了支持端侧 LLM（如 Qwen-1.5-0.5B / Yi-6B-2Bit 量化）而无需承担极其昂贵的 CDN 下行带宽费用，引入纯前端 P2P 机制：

#### 4.10.1 核心机制
- **去中心化 CDN 替代**：通过集成 WebTorrent，当首批玩家（种子）通过传统轻量化 CDN 拉取到大模型权重文件（通常 >1GB）后，后续新玩家加载游戏时，直接通过 WebRTC 从当前在线的高吞吐量玩家节点拉取模型切片。
- **文件碎片校验**：将模型文件切分为细小区块，哈希锚定在不可篡改的智能合约或只读配置中保证完整性。通过浏览器 `OPFS` 分片写入并流式解压，在达到安全阈值后启动 WebLLM 推理。
- **游戏化包装（阿卡夏网络贡献）**：玩家主动开启“做种”（Seeding）模式、共享带将获得隐形的“阿卡夏网络贡献值”。此数值可增加挂机收益或降低灾变（熵增）概率。为了防止恶意消耗，默认只在设备连接 WiFi 且电量充足时进行做种同步。

### 4.11 基于 WebRTC 的局域网“无缝空间折跃”

彻底解决纯本地方案（如 OPFS/IndexedDB）跨设备游完的割裂痛点，实现比云端存档更快的数据同步：

#### 4.11.1 功能流程
- **QR 身份握手**：用户在 PC 浏览器选择“设备接力/多端同步”，生成一个携带 WebRTC SDP 信息的动态二维码。手机浏览器扫码后，两端基于局域网或公网 STUN 穿透建立高带宽 P2P 直连。
- **高速全量/增量克隆**：以最高百兆每秒的速度，将 PC 端的 SQLite Wasm 数据库文件（包含市民、记忆、联邦状态）全包发送至手机 OPFS 中。完全不涉足任何云端中转服务器。
- **接力模拟**：手机端加载接收到的存档后即刻进行接力计算。玩家在手机进行干预后，同理传回PC。

### 4.12 Service Worker “幽灵时间”后台结算（Periodic Background Sync）

在 PWA 模式下实现真正的“休眠态宇宙演算”：

#### 4.12.1 实现机制
- **API 集成**：借助 `Periodic Background Sync` API。当用户安装了 PWA，即使完全关闭了游戏界面，只要设备处于充电并连接 WiFi 的理想状态，系统会自动允许 Service Worker 每隔数小时唤醒一次（数秒至十数秒不等）。
- **幽灵演算**：在这短暂的唤醒窗口期，执行高度简化的宏观统计推演逻辑（甚至离线校验一次市容大事件）。结算在后台悄然进行并写入 OPFS。待下次用户打开页面（进入极速预热），文明已在后台“自行流转”了百年。

## 3. 功能模块需求（V13.0 补充）

### 3.28 强硬核生物识别加密系统（WebAuthn / Passkeys）

为数字生命注入不可复制的“物理之魂”：

#### 3.28.1 设计痛点与方案
- 在无后端应用中，数字资产与配置极大依赖本地存储。传统密码容易被恶意脚本窃取。
- **方案**：使用 WebAuthn API，调起设备的 TPM / Secure Enclave（如 TouchID, FaceID, Windows Hello）。这产生的非对称私钥永远不会离开设备的物理安全模块。
- **玩法融合**：玩家在注册创世纪或确认最高级 DAO 法案时，按指纹验证作为“神启（God Touch）”。指纹生成的秘钥将用于加密以太坊钱包私钥和重要的 OPFS SQLite 加密段（如 SQLCipher for Wasm），实现史诗级的物理数字双重确权。

### 3.29 多模态全景交互体验深化

无需依赖网络调用的极客外设体验：

#### 3.29.1 Web Speech API (零延迟静默发声)
- 利用 `window.SpeechRecognition` (基于系统本地库) 和 `speechSynthesis`，允许玩家用自然语言对着屏幕下达神力指令（如：“降下甘霖”、“批准法案”），并在终端收到模拟市民毫无延迟的合成音频效忠。

#### 3.29.2 A11y “盲眼先知”模式（硬核听觉玩法）
- 不仅仅作为无障碍特性：利用极高质量的 ARIA 树结构，允许玩家关闭显示器，以闭眼状态进行游玩。
- 系统纯粹通过 Screen Reader 和 Web Audio API 的 Spatial Audio（空间音频）来呈现粒子的崩溃、市民的请愿。玩家成为真正的“盲眼先知”，依靠听音辨位来操作数字城邦。

#### 3.29.3 Gamepad API 四维触觉反馈
- 除了眼动追踪，新增外设（以手柄为例）读取接入控制。所有“熵增”事件、市民大规模暴动、系统算力波动、反应堆熔毁危机，均映射为手柄的高低频震动马达反馈。打破虚拟与现实物理触觉的第四面墙。

### 3.30 WebGPU Compute 实时全局光照计算（极致视觉）

为了将《永夜熵纪》的赛博朋克深渊感推向 3A 级别：

#### 3.30.1 实现路径
- 在原本基于实例化（Instanced Voxel）的渲染基础上，利用 WebGPU Compute Shader 手写轻量级的 SVO（稀疏体素八叉树）进行光线步进（Raymarching）。
- 在旗舰级设备并且选择了“极致性能模式”时，所有的汉字粒子群将作为局部动态光源相互投射全动态阴影和全局光照（Global Illumination），实现纯血版光线追踪级的浏览器“文字海”视觉奇观。
